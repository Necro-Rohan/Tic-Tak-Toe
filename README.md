# Tic-Tac-Toe

A simple command-line Tic-Tac-Toe game built using Python. Play against a friend in a turn-based match!

## Features
- Two-player mode
- Simple number-based board selection
- Detects winning conditions and declares the winner
- Handles invalid moves and occupied cells

## How to Play
1. Each player takes turns choosing a number (0-8) corresponding to a cell on the board.
2. The first player is assigned **X**, and the second player is assigned **O**.
3. The game ends when a player wins by aligning three marks in a row, column, or diagonal, or when the board is full (Draw).
